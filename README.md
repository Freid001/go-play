# go-play
A simple go lang demo for beginners.

## Installation
```bash
go get github.com/freid001/go-play
```

## Useage

```bash
// run
go run *.go

// hello
go run *.go hello -name="your_name"

// time
go run *.go time

// guess Number
go run *.go guessnumber

// fizzbuss
go run *.go fizzbuzz -numbers="comma_separated_number_string"
```
